---
title: Group tags
---

Tag | Description
----|------------
`admin_age_verification_enabled` | For groups that were part of the Age Verification Beta. (Deprecated with the public release of the feature)
`admin_hide_member_count` | Hides both online members and total number of members in the group. Used for the [VRCHAT.0000](https://vrc.group/VRCHAT.0000) staff group.
`admin_featured_events_enabled` | Certain endorsed groups may enable the `featured` flag when creating a calendar event.
`admin_vrc_event_group_fair_enabled` | Certain endorsed groups may use the `vrc_event_group_fair` calendar event tag.
